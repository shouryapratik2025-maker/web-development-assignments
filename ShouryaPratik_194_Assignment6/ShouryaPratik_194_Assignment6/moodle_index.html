📝 Assignments

01

HTML Basics – Build a Personal Profile Page

Create a simple HTML webpage that includes a heading, paragraph, image, and a list of your hobbies.

📅 Due Date: November 15, 2025

02

CSS Styling – Style Your Profile Page

Apply CSS to your Assignment 1 page. Add colors, fonts, margins, padding, and a hover effect on at least one element.

📅 Due Date: November 22, 2025

03

JavaScript Interactivity – Course Selection Page

Build a course selection portal using HTML, CSS, and JavaScript that allows a student to select a course and display it on screen.

📅 Due Date: December 1, 2025

04

Moodle Page – Responsive Course Dashboard

Design a Moodle-inspired course page with navigation, announcements, assignments, and resources using HTML, CSS, and JavaScript.

📅 Due Date: December 10, 2025
