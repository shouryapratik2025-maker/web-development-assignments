👋 About This Website

A little bit about this project and how it was made

What is This Website?

Delicious Recipes is a cooking recipes website created as part of a first year Web Development assignment. The goal was to build a clean, multi-page website using only HTML and CSS — no JavaScript was used anywhere in this project.

What Technologies Were Used?

This website was built using HTML5 for the structure and CSS3 for all the styling and layout. Key CSS features used include Flexbox for the navigation bar and card layouts, CSS Grid for the recipe cards section, hover effects on buttons and cards, and media queries to make the site responsive on mobile devices.

What Pages Does the Website Have?

The website contains a home page with featured recipe cards, five individual recipe pages (Creamy Garlic Pasta, Fluffy Pancakes, Greek Salad, Chicken Biryani, and Chocolate Brownies), and this About page — making a total of 7 pages.

About the Developer

This website was created by a first year web development student as part of a structured assignment to learn HTML and CSS. It demonstrates skills in semantic HTML, CSS layout techniques, responsive design, and multi-page website structuring.

How to Deploy on GitHub Pages?

To publish this website, the folder was uploaded to a GitHub repository. Then, in the repository settings under the Pages section, the source was set to the main branch. GitHub then automatically generates a live URL for the website, which can be shared with anyone.
